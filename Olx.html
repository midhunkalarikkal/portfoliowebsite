OLX Clone, a MERN stack application designed to strengthen full-stack development concepts through CRUD operations. This project integrates React.js for the front end, Express.js for the backend, MongoDB for database management, and Node.js to handle server-side logic. It focuses on state management using Context API, efficient styling with Tailwind CSS, and smooth notifications via React Hot Toast. For authentication and hosting, Firebase is integrated, ensuring a responsive and seamless user experience. Additionally, I implemented Cloudinary for efficient image storage. The project includes key features like CRUD Operations, User Authentication, Responsive UI, State Management, and Firebase Hosting, making it a well-structured and scalable application.
