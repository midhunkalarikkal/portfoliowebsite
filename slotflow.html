I am currently developing an Appointment Booking Application from scratch using the MERN Stack, TypeScript, TanStack Query, and shadcn/ui for a seamless user experience. The project features multiple user roles, ensuring tailored access and functionality for admins, service providers, and customers. To optimize scalability and performance, I have designed a well-structured database with over 12 schemas, leveraging AWS S3 for secure file storage. The system is built with efficient state management and real-time data fetching using TanStack Query.
