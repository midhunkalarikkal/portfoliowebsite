LapshoP Ecommerce

Project information

  • Category: Node Web Application
  • Project date: 5 June, 2024
  • Project URL: https://github.com/midhunkalarikkal/lapshop

I developed a feature-rich eCommerce web application using Node.js, Express.js, and MongoDB, designed to be robust and scalable. The application includes a dynamic front-end built with EJS, HTML5, CSS3, Bootstrap, jQuery, and FontAwesome, ensuring a seamless and responsive user experience. Hosted on AWS with Nginx for efficient server management, the project provides comprehensive functionalities for both users and administrators. Users can enjoy features like coupons, wallets, referral programs, product returns, wishlists, and more, while administrators have access to a dashboard with real-time information, tables, and charts. The admin panel also supports account management, image uploads for categories, products, and homepage ad-banners. With Git for version control and a variety of npm packages, this project reflects my commitment to delivering complete and production-ready web solutions.
