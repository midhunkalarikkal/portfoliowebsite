I developed a Netflix Clone using React, Tailwind CSS, and Redux, integrating Firebase for authentication and hosting. This project leverages the TMDB API to fetch movie data and employs custom hooks to efficiently handle different category-based requests. To enhance user experience, I implemented shimmer effects for smooth content loading and various custom features for seamless navigation. Authentication is secured with Firebase, allowing users to register, log in, and reset passwords via email verification. The platform is fully responsive, delivering an optimized viewing experience across all devices. Additionally, I integrated a Gemini 1.5 Flash-powered movie recommendation system with rate limiting, ensuring personalized suggestions while maintaining system performance.
